#### Installation

##### Prerequisites
exoTras requires python >= 3.7.

    "numpy", "pandas", "scipy",
    "scanpy", "statsmodels", "gseapy"

##### Prerequisites
```bash
pip install exoTras
```
We also suggest to use a separate conda environment for installing exoTras.
```bash
conda create -y -n exoTras_env python=3.7
conda activate exoTras_env
pip install exoTras
```
